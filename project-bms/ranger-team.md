# Ranger Team — BMS Project

## ตำแหน่งและหน้าที่

### PM (Project Manager)
- วางแผนและติดตาม timeline ของ project
- จัดลำดับความสำคัญของ feature และ backlog
- ประสานงานภายในทีมและกับ stakeholder
- รายงานความคืบหน้าและจัดการ risk

---


### BA (Business Analyst)
- รวบรวมและวิเคราะห์ความต้องการจากผู้ใช้งาน
- เขียน requirement และ user story
- ประสานงานระหว่าง stakeholder กับทีม Dev
- ตรวจสอบว่า feature ที่ส่งมอบตรงกับความต้องการ

---

### Dev (Developer)
- พัฒนา frontend (React + Tailwind)
- เชื่อมต่อและจัดการข้อมูลผ่าน Supabase
- แก้ไข bug และปรับปรุง performance
- ดูแล deploy บน Vercel

---


### Review
- ตรวจสอบ code ก่อน merge (code review)
- ทดสอบ feature ก่อน release (QA)
- ตรวจสอบความถูกต้องของข้อมูลและ logic ทางธุรกิจ
- ให้ feedback กลับทีม Dev
