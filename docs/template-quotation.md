## เงื่อนไขทั่วไป

1. **Revision** — รวม ___ รอบแก้ไขต่อ Phase หากเกินคิดเพิ่มรอบละ ______ บาท
2. **Scope Change** — การเพิ่มฟีเจอร์หรือหน้าที่ไม่อยู่ใน Scope เดิมจะประเมินราคาและระยะเวลาเพิ่มเติม
3. **Source Code** — ลูกค้าได้รับ Source Code เต็มหลังชำระเงินครบ
4. **Warranty** — รับประกัน Bug Fix ___ เดือนหลัง Go-live (เฉพาะ Bug ที่เกิดจากการพัฒนา)
5. **Confidentiality** — ทั้งสองฝ่ายจะรักษาความลับของข้อมูลโปรเจกต์
6. **ยกเลิกสัญญา** — หากลูกค้ายกเลิกหลังเริ่มงาน จะไม่คืนเงินงวดที่ชำระแล้ว
