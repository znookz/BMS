# ระบบ Bus Management System (BMS) — Requirements

## ภาพรวมระบบ
ระบบจัดการรถบัสสำหรับใช้ภายในองค์กร รองรับการทำงานบน PC เป็นหลัก และ Mobile
- Frontend: React + Tailwind CSS
- Database: PostgreSQL ผ่าน Supabase
- Deploy: Vercel
- ภาษา UI: ไทย

## Modules

### 1. Overview Dashboard
- สรุป KPI ทุก module บนหน้าเดียว
- แสดง: จำนวนรถที่ใช้งาน / เข้าซ่อม, ค่าใช้จ่ายรายเดือน, รถที่ใกล้ถึงรอบซ่อม, ใบอนุญาตที่ใกล้หมดอายุ, complaint ที่ยังไม่แก้ไข
- กราฟ Cost Control แยกรถแอร์ vs พัดลม
- Export: PDF

---

### 2. Driver
- ข้อมูลส่วนตัว: ชื่อ-นามสกุล, เลขบัตรประชาชน, สิทธิรักษา (ประกันสังคม), เบอร์โทร, ที่อยู่
- สายรถที่ขับ (ลิงก์กับ Bus)
- กะการทำงาน
- บันทึกตรวจสุขภาพประจำปี
- บันทึกการฝึกอบรม (ลิงก์กับ Training)
- Export: Excel, PDF

---

### 3. Bus
#### รถแอร์
- ทะเบียน, ยี่ห้อ, โรงงาน, บริษัทที่รับผิดชอบ
- สถานะ: ใช้งาน / เข้าซ่อม / จำหน่ายแล้ว
- คนขับปัจจุบัน (ลิงก์กับ Driver)
- วันที่รับรถ, วันประกันเริ่ม-หมด
- ลิงก์ระบบตรวจจับพฤติกรรมการขับรถ

#### รถพัดลม — Mock
- ข้อมูลเดียวกับรถแอร์
- แยก category ชัดเจน

---

### 4. Maintenance
- บันทึกเลขไมล์รายวัน (per bus)
- แจ้งเตือนอัตโนมัติเมื่อถึง 80% ของรอบซ่อม (In-app notification)
- บันทึกการเข้าซ่อมตามรอบ: อาการ, รายการซ่อม, ค่าใช้จ่าย
- บันทึกรถเสียนอกรอบ: สาเหตุ, รายการซ่อม, ค่าใช้จ่าย
- บันทึกซ่อมจากอุบัติเหตุ (ลิงก์กับ Insurance): ความเสียหาย, รายการซ่อม, ค่าใช้จ่าย
- แนบรูปภาพได้ (jpg/png/pdf, max 10MB ต่อไฟล์)
- Export: Excel, PDF

---

### 5. Cost Control
- Input ค่าใช้จ่ายทั้งหมดที่เกี่ยวกับรถบัส
- แยกประเภท: รถแอร์ / รถพัดลม
- แสดงใน Dashboard: กราฟรายเดือน, รายปี, แยกประเภทค่าใช้จ่าย
- Export: Excel, PDF

---

### 6. Training
- บันทึกหลักสูตรทั้งหมด (ชื่อ, ระยะเวลา, ความถี่ที่ต้องอบรม)
- บันทึกการอบรมรายคน: ใครอบรมหลักสูตรไหน, วันที่, ผลการอบรม
- แจ้งเตือนเมื่อใบรับรองใกล้หมดอายุ (In-app, 30 วันล่วงหน้า)
- Export: Excel, PDF

---

### 7. License & Insurance
- เก็บข้อมูลใบอนุญาตต่างๆ (พรบ, ประกันภัย, ใบขับขี่คนขับ)
- แจ้งเตือนก่อนหมดอายุ (In-app, 30 วันล่วงหน้า)
- บันทึกเคส/เคลม: วันที่, รายละเอียด, ค่าใช้จ่าย, สถานะการเคลม
- ข้อมูลเคลมใช้วิเคราะห์การต่อเบี้ยประกันปีต่อไป
- แนบรูปภาพได้ (jpg/png/pdf, max 10MB ต่อไฟล์)
- Export: Excel, PDF

---

### 8. Customer Complaint
- บันทึกข้อร้องเรียน: วันที่, ผู้ร้องเรียน, รายละเอียด, รถ/คนขับที่เกี่ยวข้อง
- บันทึกการแก้ไข, สถานะ (รอดำเนินการ / ดำเนินการแล้ว)
- Export: Excel, PDF

---

### 9. Safety Measurement — Mock
- Pre-trip Inspection: checklist รายวันก่อนออกรถ (ไฟ, เบรก, ยาง, แอร์)
- บันทึกผลทดสอบแอลกอฮอล์คนขับ (รายวัน)
- บันทึก Near-miss incidents
- บันทึกการซ้อมความปลอดภัยประจำเดือน
- Log การฝ่าฝืนความเร็ว (ลิงก์จากระบบตรวจจับพฤติกรรม)
- Export: Excel, PDF

---

### 10. Purchase
- รายการสั่งซื้ออุปกรณ์: รถแอร์ และ สถานีน้ำมัน
- ข้อมูล: วันที่สั่ง, รายการ, จำนวน, ราคา, สถานะ (รอของ / รับของแล้ว)
- Export: Excel, PDF

---

### 11. Fuel (ATG)
- แสดงข้อมูลเติมน้ำมันรายวัน real-time จากระบบ ATG
- แยกตาม: บริษัทรถ, โรงงาน, รถ
- Dashboard: ปริมาณน้ำมัน vs ค่าใช้จ่าย รายวัน/รายเดือน

## ฟีเจอร์ระบบ (System Features)

### Notification (In-app)
| เหตุการณ์ | เวลาแจ้งเตือน |
|---|---|
| รถใกล้ถึงรอบซ่อม | เมื่อถึง 80% ของระยะ |
| ใบอนุญาต/ประกันใกล้หมดอายุ | 30 วันล่วงหน้า |
| ใบรับรองการอบรมใกล้หมดอายุ | 30 วันล่วงหน้า |
| Complaint ยังไม่ได้รับการแก้ไข | ทุก 7 วัน |

### Export
- รองรับ **Excel (.xlsx)** และ **PDF** ทุก module
- กำหนด date range ก่อน export

### File Attachment
- รองรับ: jpg, png, pdf
- ขนาดสูงสุด: 10MB ต่อไฟล์
- ใช้ใน: Maintenance, License & Insurance

### Audit Log
- บันทึกทุก action: สร้าง, แก้ไข, ลบ
- เก็บข้อมูล: user, timestamp, field ที่เปลี่ยน, ค่าก่อน/หลัง
- ดูได้ใน User Management (Admin only)
